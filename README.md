О чем проект:Данный проект представляет собой разработку, созданную для выполнения учебных и практических задач.

Основная цель проекта — продемонстрировать работу приложения, его функциональность и применение современных технологий разработки.

Проект позволяет пользователю взаимодействовать с системой через удобный интерфейс и выполнять необходимые действия в рамках поставленной задачи.

Как опробовать:проект можно опробовать по специальной ссылке или же на форуме Replit.

Какие технологии используются:используется современный интерфейс искусственных интеллектов,симуляция солнечной системы,технология классификация опасности астероидов,космические темы

Авторы:Каражанов Арнат,Куанышбай Расул,Конилбай Динмухамед
